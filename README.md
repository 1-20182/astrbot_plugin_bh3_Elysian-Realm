# 🌟 CCB插件 - 飞翔的死猪重构版 🌟

欢迎使用超可爱的CCB插件！这是一个专为QQ群设计的趣味互动插件，让你和群友们一起享受赛博世界的快乐时光～

## 🎉 功能介绍

- **CCB互动**：和群友发生赛博sex，随机生成时长和注入量
- **记录查询**：查看自己或他人的CCB记录
- **好感度系统**：根据累积注入量计算好感度
- **淫乱度系统**：根据累积注入量计算淫乱度
- **群聊排行榜**：展示当前群聊的CCB次数排行榜

## 📝 使用方法

### 基本命令

- `/ccb` - 和自己CCB
- `/ccb @某人` - 和指定群友CCB
- `/ccb查询` - 查询自己的CCB记录
- `/ccb查询 @某人` - 查询指定群友的CCB记录
- `/好感度查询` - 查询自己的好感度
- `/好感度查询 @某人` - 查询指定群友的好感度
- `/淫乱度查询` - 查询自己的淫乱度
- `/淫乱度查询 @某人` - 查询指定群友的淫乱度
- `/ccb排行` - 查看当前群聊的CCB排行榜

## 🌈 特色功能

- **随机数据**：每次CCB都会生成随机的时长和注入量，充满惊喜！
- **头像展示**：查询记录时会显示用户头像，更加生动有趣
- **群聊分离**：排行榜按群聊分离，每个群都有自己的专属排行
- **数据持久化**：所有记录都会保存，不用担心数据丢失
